# Paper Reproduction Tables

These files are compact, tracked summaries for the paper tables. They are not a
replacement for the raw benchmark directories; they record the matched rows used
in `method.tex` so a reader can verify the reported numbers without committing
large generated graphs and alignments.

`matched_full_assembly_summary.csv` restricts the cached MQLib result to the
same MG split used by the DyNACO full-assembly rows: seeds `00001` through
`00008`, five test sequences per seed, for forty matched sequences.
